# Real-Time Vehicle Counting and Classification

A computer vision-based traffic monitoring system that detects, tracks, counts, and classifies vehicles from video streams using YOLO and OpenCV.

## Features
- Real-time vehicle detection
- Vehicle counting
- Vehicle classification
- Traffic analytics
- Video stream processing

## Technologies
- Python
- OpenCV
- YOLO
- NumPy

## Applications
- Smart traffic management
- Transportation analytics
- Intelligent transportation systems
